#  Simple test Comptalib - Node.js Rest Apis with Express, Sequelize & MySQL.

## Project setup
```
npm install
```

### Run
```
node server.js
```

## MYSQL USER
```
CREATE USER 'comptalib'@'localhost' IDENTIFIED BY 'password';

GRANT ALL PRIVILEGES ON comptalib.* TO 'comptalib'@'localhost';

FLUSH PRIVILEGES;

```
